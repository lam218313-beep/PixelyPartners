## ARCHITECTURE HIGHLIGHTS

**Design Patterns Implemented:**
- Abstract Base Class Pattern (BaseAnalyzer)
- Registry Pattern (dynamic module loading)
- Async/Await Pattern (parallel execution)
- Factory Pattern (module instantiation)
- Three-Tier Fallback Pattern (path resolution)

**Key Features:**
- Single-client native (no multi-client branching)
- Asynchronous execution (10x parallel speedup)
- Modular design (add/remove modules easily)
- Standard JSON output format (all modules)
- Error handling (exceptions captured, never raised)
- Environment flexibility (local/Docker/cloud)

## QUICK START

### For Local Development

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Configure (edit .env with your OpenAI API key)
# OPENAI_API_KEY=sk-...

# 3. Terminal 1: Run orchestrator
python orchestrator/analyze.py

# 4. Terminal 2: Run frontend
streamlit run frontend/app.py

# Open: http://localhost:8501
```

### For Docker

```bash
# Build and start all services
docker-compose up --build

# Open: http://localhost:8501

# Stop
docker-compose down
```

### For Validation

```bash
# Validate everything
python validate.py

# Run tests
pytest tests/ -v
```
